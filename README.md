# Institutional Global Macro & Fixed-Income AI Terminal

## Project Overview
This production-grade quantitative finance platform streams live global market data, constructs real-time sovereign yield curves, parses Federal Reserve natural language sentiment, and generates autonomous macroeconomic execution signals.

Designed to mirror the capabilities of institutional trading desks, the terminal processes live data from the Federal Reserve Economic Data (FRED) API and global exchanges to evaluate cross-asset risk, macroeconomic surprises, and portfolio revaluations.

---

## 🏗️ Core Architecture & Features

The platform is structured into modular execution environments, allowing risk managers and traders to rapidly scan global asset classes and deep-dive into fixed-income theory.

### 1. ⚡ Pre-Market Tear Sheet (Momentum Matrix)
* Scans global Equities, Sovereign Bonds, FX, and Commodities.
* Computes real-time 1-Day, 1-Week, and 1-Month percentage momentum vectors.
* Applies automated conditional heatmap formatting for instant regime identification (e.g., Risk-On vs. Flight-to-Safety).

### 2. 🏛️ Fixed-Income Engine & Yield Curve Visualizer
* **Live Curve Generation:** Constructs a continuous U.S. Sovereign Yield Curve using Cubic Spline Interpolation
